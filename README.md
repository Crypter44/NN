# NN Library
This is a simple neural network library implemented in Python. 
It provides basic functionalities to create, train, and evaluate neural networks. 
It is inspired by popular libraries like PyTorch and implemented for educational purposes.
As it only uses NumPy, it is not optimized for performance.

## Usage
To use this library, simply clone the repository, install the required dependencies, and import the necessary modules in your Python script.

For examples and documentation, refer to the individual classes or look at the tests provided in the `test` directory.
